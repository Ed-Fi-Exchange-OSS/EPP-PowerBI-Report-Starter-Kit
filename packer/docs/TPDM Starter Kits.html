This Virtual Machine Contains the Ed-Fi ODS / API & TPDM Starter Kits.

The purpose of this virtual machine is to showcase the Clinical Experience & Performance and Program Diversity & Persistence TPDM Starter Kits.


Ed-Fi Power BI Assets Installed

EPP Performance Power BI Starter Kit

Open the EPP Performance Starter Kit by going to the virtual machine's desktop and clicking on the shortcut titled "Ed-Fi EPP Performance" as depicted below.

Ed-Fi EPP Diversity and Completion Power BI Starter Kit

Open the Ed-Fi EPP Diversity and Completion Starter Kit by going to the virtual machine's desktop and clicking on the shortcut titled "Ed-Fi EPP Diversity and Completion" as depicted below.


Once Power BI opens, locate the Modeling menu from the top toolbar. Then proceed to click on "View as".

Use the following demo user accounts to experience what this dashboard would look like for different roles.

*Note: You will need to click the option that corresponds to the type of user you want to view the dashboard as - "Standard" for the "Program Manager Role" or "Supervisor" for the "Dean" role.
